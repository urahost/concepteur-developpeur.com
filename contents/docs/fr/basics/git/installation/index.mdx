# Installation de Git

L'installation de Git varie selon votre système d'exploitation. Ce guide couvre les méthodes d'installation pour les plateformes les plus courantes.

## Sous Windows

Pour installer Git sous Windows, vous pouvez [le télécharger ici](https://git-scm.com/download/win).

<Note type="info">
L'installateur Windows inclut un client GUI et des outils en ligne de commande. Pendant l'installation, vous pouvez configurer Git pour qu'il fonctionne avec votre terminal et éditeur de texte préférés.
</Note>

## Sous Mac (Homebrew)

Pour Mac, utiliser **Homebrew** est l'approche recommandée :

```sh
brew install git
```

<Note>
Si vous n'avez pas Homebrew installé, vous pouvez l'obtenir depuis [brew.sh](https://brew.sh) ou utiliser l'installateur officiel Git depuis [git-scm.com](https://git-scm.com/download/mac).
</Note>

## Sous Linux

Pour Linux, l'installation dépend de votre gestionnaire de paquets (en fonction de votre distribution).

### Avec APT (Ubuntu/Debian)

```sh
sudo apt-get install git
```

### Avec Pacman (Arch Linux)

```sh
sudo pacman -S git
```

<Note type="success">
Après l'installation, vérifiez que Git est correctement installé en exécutant `git --version` dans votre terminal. Cela affichera la version de Git installée.
</Note>

## Prochaines étapes

Une fois Git installé, vous pouvez apprendre le [versioning avec Git](/docs/basics/git/versioning) pour comprendre comment Git suit les modifications dans vos projets.
