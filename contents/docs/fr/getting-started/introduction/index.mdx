Bienvenue dans la **documentation de formation CDA** ! Cette plateforme regroupe l'ensemble des ressources pédagogiques pour la formation Concepteur Développeur d'Applications de Valenciennes P2. Vous y trouverez des guides pratiques, des références techniques et des bonnes pratiques pour votre parcours de développement.

## À propos de cette formation

La formation **Concepteur Développeur d'Applications** (CDA) est une formation professionnalisante qui vous prépare aux métiers du développement web et logiciel. Cette documentation accompagne votre apprentissage avec des contenus structurés et des exemples pratiques.

<Note title="Navigation">  
Utilisez la barre latérale pour naviguer entre les différentes sections. Chaque module est organisé de manière progressive pour faciliter votre apprentissage.
</Note>

## Contenu de la formation

Cette documentation couvre les **compétences essentielles** du développeur moderne. Les principales sections incluent :

- **Bases du développement** : Git, méthodologies agiles, gestion de projets
- **Qualité du code** : Conventions, linting, clean code
- **Technologies modernes** : Frameworks, bibliothèques, outils de développement

### Compétences développées

| **Domaine**                     | **Compétences acquises**                                |
|----------------------------------|---------------------------------------------------------|
| Gestion de version              | Git, branches, collaboration, workflows                |
| Méthodologies                   | Agile, Scrum, planning poker, vélocité                 |
| Qualité du code                 | Conventions, linting, tests, documentation             |
| Outils de développement         | IDE, débogage, profiling, monitoring                   |
| Architecture logicielle         | Patterns, conception, modularité                       |
| Développement web               | Frontend, backend, API, bases de données               |

## Organisation du contenu

Cette documentation est structurée en modules progressifs :

- **Fondamentaux** : Git, méthodologies, gestion de projet
- **Qualité** : Conventions de code, outils de linting, bonnes pratiques
- **Conception** : Architecture logicielle, patterns, modélisation
